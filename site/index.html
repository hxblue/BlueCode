能力概览

从本地终端到协作式 Agent

01

工具调用

内置读写、搜索、Shell、MCP 工具注册, 支持流式展示调用过程与结果。

02

权限引擎

按默认、编辑、计划、绕过等模式控制工具执行, 让高风险操作有明确边界。

03

SubAgent

把复杂任务拆给子 Agent, 每个任务有独立上下文、工具过滤和可恢复会话。

04

Team + Worktree

用独立 worktree 承载多成员协作, 通过任务列表和邮箱协调长期工作。

Run locally

本地运行

git clone https://github.com/hxblue/BlueCode.git
cd BlueCode
./gradlew shadowJar
java -jar build/libs/bluecode.jar

需要 JDK 21。复制 `.bluecode/config.yaml.example` 为 `.bluecode/config.yaml` 后填入自己的 provider 配置。

公开展示站只播放预录制 demo, 不会读取你的本地文件, 也不会调用真实模型。
